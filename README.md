## 🎮 Games

### Tetris
A classic block-stacking puzzle game built entirely with vanilla HTML/CSS/JS.

- All 7 tetrominoes (I, O, T, S, Z, J, L) with rotation
- Ghost piece preview, next piece display
- Score system with level progression (speed increases)
- High score saved locally
- Keyboard controls (arrow keys + space for hard drop)
- Touch controls and swipe gestures for mobile
- Pause/resume with P key

**Play now:** [html/tetris.html](https://ckw19810413.github.io/qwen3.6-27b-tiny-games/html/tetris.html)

### Space Shooter
A classic space shooter game built entirely with vanilla HTML/CSS/JS.

- Player spaceship with left/right movement
- Enemy waves with different types
- Health bar and score tracking
- Keyboard controls (← → ↑ ↓)
- Touch controls for mobile

**Play now:** [html/space-shooter.html](https://ckw19810413.github.io/qwen3.6-27b-tiny-games/html/space-shooter.html)

## 🧠 How It Works

All games are single-file HTML applications generated via LLM-assisted coding using [Qwen3.6-27B](https://huggingface.co/Qwen/Qwen3.6-27B). Each game runs entirely in the browser with no server required — just open the file and play.
