# Qwen3.6-27B Tiny Games

Pure HTML/CSS/JS games generated by [Qwen3.6-27B](https://huggingface.co/Qwen/Qwen3.6-27B). Run directly in browser — no build step, no dependencies.

## Games

### Space Shooter
A classic space shooter game with:
- Player spaceship with left/right movement
- Enemy waves with different types
- Health bar and score tracking
- Keyboard controls (← → ↑ ↓)
- Touch controls for mobile

**Play:** [html/space-shooter.html](html/space-shooter.html)

## How it works

All games are single-file HTML applications generated via LLM-assisted coding. Each game runs entirely in the browser with no server required.
